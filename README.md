## 🎯 **Agente de Ventas Inteligente para NGX**

El **NGX Voice Sales Agent** es un agente de ventas conversacional especializado que conoce profundamente los servicios, programas y audiencia de NGX. Este agente único (no es un sistema multi-agente) transforma cualquier punto de contacto digital en una experiencia de ventas personalizada y efectiva. Con **una sola línea de código**, convierte landing pages, lead magnets, blogs y aplicaciones en poderosas herramientas de conversión para vender los programas NGX.

## 🚀 **Quick Start Guide**

### **1. Desarrollo Local**

```bash
# Clonar repositorio
git clone https://github.com/270aldo/agent.SDK.git
cd agent.SDK

# Backend Setup
python run.py --host 0.0.0.0 --port 8000

# Frontend SDKs
cd sdk && npm run build

# Testing Completo
./run_tests.sh coverage
```

### **2. Variables de Entorno Requeridas**

```env
# APIs de Terceros
OPENAI_API_KEY=your_openai_api_key
ELEVENLABS_API_KEY=your_elevenlabs_api_key

# Base de Datos
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key

# Autenticación y Seguridad
JWT_SECRET=your_secure_jwt_secret_32_characters_minimum
JWT_ALGORITHM=HS256
ALLOWED_ORIGINS=https://your-domain.com,https://staging.your-domain.com

# Rate Limiting
RATE_LIMIT_PER_MINUTE=60
RATE_LIMIT_PER_HOUR=1000

# Aplicación
DEBUG=True
LOG_LEVEL=INFO
ENVIRONMENT=development
```

### **3. Deployment Production**

```bash
# Docker Production
docker-compose -f docker/docker-compose.yml up --build

# Testing de Seguridad
./run_security_tests.sh

# Health Check
curl https://your-api.com/health
```

## 📞 **API Endpoints Principales**

La API estará disponible en `http://localhost:8000` con los siguientes endpoints:

### **Core Conversation API**
- **GET /health**: Verificar que la API está funcionando
- **POST /conversations/start**: Iniciar nueva conversación con contexto
- **POST /conversations/{id}/message**: Enviar mensaje a conversación
- **GET /conversations/{id}/audio**: Obtener audio de respuesta
- **POST /conversations/{id}/end**: Finalizar conversación

### **Revolutionary Features API**
- **POST /triggers/analyze**: Analizar comportamiento para triggers
- **GET /ai/context/{touchpoint}**: Obtener configuración IA contextual
- **POST /analytics/track**: Enviar eventos de engagement
- **GET /voice/avatar/states**: Estados disponibles del avatar

Para documentación completa: `http://localhost:8000/docs`
